# Xiaochuan Li

MSc Human & AI · Causal NLP · Mechanistic Interpretability · Stable Inference

I study how language models preserve or fail to preserve stable reasoning mechanisms under perturbation, distribution shift, and intervention.

## Research Focus

- Stable inference under perturbation
- Shortcut learning and reasoning failure
- Mechanism shift in language models
- Controlled reasoning environments
- Latent control states and prompt framing
- Causal Inferential Yield (CIY)

## Current Projects

| Project | Description |
|---|---|
| PJ1: Inferential Yield | A framework for measuring reasoning as stable conclusion generation under information constraints |
| PJ2 | Mechanistic interpretability project on prompt framing and latent control-state shifts |
| PJ3: Perturbation & Failure Analysis | Controlled perturbation experiments, shortcut reversal, and reasoning failure taxonomy |


## Current Research Thesis

> Reasoning is not correctness.  
> Reasoning is the preservation of stable mechanisms under perturbation and intervention.
